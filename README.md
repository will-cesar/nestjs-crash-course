## <a name="installation"></a> Installation

```bash
$ npm install
```

## <a name="running-the-app"></a> Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## <a name="test"></a> Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
